La aplicacion es bastante simple, solo se compone de un area para el texto y un boton para
generar el arbol AST del codigo ingresado.

Los modulos que se soportan se encuentran el archivo entradas.txt. Aunque se puede hacer cualquier tipo de variacion a esos modulos teniendo en cuenta que el simbolo - debe tener un espacio si viene 
un numero posterior a el. Finalmente solo utilizar estas comillas (") debido a que no se reconocen 
otro tipo de ellas.
